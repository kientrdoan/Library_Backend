1.	INSTALL PACKAGES
	RUN COMMAND: pip install -r requirements.txt

2	CONFIG IP ADDRESS FOR CALLING API (AT COM.LIBRARY.SERVICE.API)
	- 10.0.2.2 FOR VIRTUAL DEVICE.
	- IP_WIFI FOR REAL DEVICE. (GET BY RUN COMMAND 'ipconfig' AT CMD OF WINDOWS)

3.	RUN SERVER LOCALHOST
	3.1 FOR VIRTUAL DEVICE:
    	- RUN COMMAND: python manage.py runserver
   	3.2 FOR REAL DEVICE:
		- CONNECT TO THE SAME WIFI.
       	- RUN COMMAND: python manage.py runserver 0.0.0.0:8000
